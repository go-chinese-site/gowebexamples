通过例子学习 Go Web 编程

Go 是一种用于构建简单、快速和可靠软件的开源编程语言。 看看 这里，那些伟大的公司使用 Go 提供他们的服务。

Go Web 示例 提供了易于理解的代码片段, 关于如何在 go 编程语言中进行 web 开发。 它受 Go By Example 启发, 它对于了解该语言的基础知识有很大帮助。

如果你正在学习 Web 编程, 或者只是开始, 在这里你可以找到一些好的例子和教程。我们的目标是提供非常详细、简洁的例子, 以便你可以成为下一个 Go web 网站开发者！Go Web 示例涵盖了 web 编程的基本知识。 从路由和模板到中间件和 websockets。在这里，您可以找到详细、简洁的代码片段教程。

让我们从第一个例子开始，如何创建一个经典的 "Hello World" web 应用程序或直接转到Routing (使用 gorilla/mux 路由)。
